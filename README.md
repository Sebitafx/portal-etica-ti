# Portal de Ética y Buenas Prácticas en el Campo Laboral del Desarrollo de Software

Este proyecto es una plataforma web interactiva diseñada para proveer a profesionales y estudiantes de TI de una guía clara, casos de estudio técnico-éticos y un *checklist* de seguridad. El objetivo es fomentar una cultura de prevención y responsabilidad en los entornos de desarrollo y trabajo.
